# babel-preset-blueflag
A preset for Blue Flag's oppinions on babel plugins

## Installation

```
npm install --save babel-preset-blueflag
```

```js
// Place this in your configs
{
  presets: ['blueflag']
}

```

## Plugins & Presets
```
babel-plugin-transform-runtime
babel-preset-es2015
babel-preset-es2016
babel-preset-es2017
babel-preset-react
babel-preset-stage-3
 ```
